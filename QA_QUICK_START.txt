## 💡 PRO TIPS

### Tip 1: Quick Health Check
```bash
# Check if backend is responding
curl http://localhost:8080/api/products
# Response: JSON array of products = ✅ Working
```

### Tip 2: View API Requests
```
1. Open browser
2. Press F12 (Developer Tools)
3. Go to Network tab
4. Perform action
5. See all API calls with responses
```

### Tip 3: Check JWT Token
```
1. Press F12 in browser
2. Go to Application tab
3. Check localStorage
4. Look for 'authToken' key
5. Should contain long JWT string
```

### Tip 4: Run Tests in Background
```bash
# Run tests and save results
node api-test.js > test-results.txt 2>&1
# Check results later: type test-results.txt
```

### Tip 5: Database Queries
```bash
# Connect to database
psql -U postgres -h localhost -d ceylon_mango

# Check tables
\dt

# Query data
SELECT COUNT(*) FROM users;
SELECT * FROM products;
```

## 🐛 QUICK TROUBLESHOOTING

| Issue | Quick Fix | More Help |
|-------|-----------|-----------|
| **Backend won't start** | Check PostgreSQL running | TESTING_GUIDE.md §Troubleshooting |
| **Frontend won't start** | `npm install` then `npm run dev` | SETUP_GUIDE.md |
| **CORS errors** | Verify CorsConfig.java | CONFIGURATION_SUMMARY.md |
| **Port 8080 in use** | Kill Java process | Task Manager |
| **Tests fail** | Ensure all services running | TESTING_GUIDE.md |
| **JWT errors** | Check localStorage | Browser DevTools |
| **Compile errors** | Run `mvn clean install` | backend/README.md |
